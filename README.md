# Spam SMS Classifier

This project uses Natural Language Processing (NLP) and Machine Learning to classify SMS messages as Spam or Not Spam.

## Technologies Used
- Python
- TF-IDF Vectorizer
- Multinomial Naive Bayes
- scikit-learn

## Dataset
SMS Spam Collection Dataset (Kaggle)

## Description
Text messages are converted into numerical features using TF-IDF.  
A Naive Bayes classifier is trained to identify spam messages with ~96% accuracy.
